状态：`PENDING USER VERIFICATION`

## 前置样本

- 书架中准备一本已阅读到中间章节、已设置分组/排序、自定义封面或备注、单书显示配置的网络书。
- 至少两个启用书源能搜索到书名相同的候选，其中一个作者匹配、一个作者不匹配更佳。
- 准备一个目标候选已作为另一条书架记录存在的主键冲突场景。
- 普通规则样本和 JavaScript 样本分别执行；敏感账号、Cookie 和书源正文不写入记录。
