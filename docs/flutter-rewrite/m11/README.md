## 当前 Feature

### 1. 整书换源（已实现，等待验收）

- 从书籍详情、书架单选和文本阅读器进入。
- 在当前启用书源中有界并发搜索同名候选，可选作者校验和书源范围。
- 候选必须完成详情和完整目录加载后才允许确认。
- 原子替换旧书主键、旧目录、新书和新目录，并在事务内复查目标主键冲突。
- 可选择迁移阅读进度、分组排序、自定义封面、分类标签、备注简介和单书阅读/显示配置。
- 阅读器入口换源前保存稳定进度，成功后使用新主键替换当前阅读路由。

### 2. 单章换源（已实现，等待验收）

只替换阅读器当前章节的正文，不改变书籍主键；复用整书换源的多源搜索基础设施，新增候选目录
模糊匹配定位目标章节。详见 [专项记录](./chapter_change_source/README.md)。

### 3. 离线下载（已实现前台运行范围，等待验收）

阅读器内下载队列，数据库持久化、有界并发、失败重试；明确不包含 Android 式前台服务/通知，
应用被系统回收或退出后下载会停止。详见 [专项记录](./offline_download/README.md)。

明确不包含（三个 Feature 共同边界）：自动换源、批量换源、书源置顶/禁用/删除、旧正文缓存批量
删除、跨书全局缓存管理仪表盘、离线下载的前台服务/通知/暂停恢复。

AI 没有运行 Flutter、Dart、Gradle、Xcode、构建、测试、分析、格式化或应用启动命令。
